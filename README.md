# 中国海洋大学硕士博士学位论文 LaTeX 模板（理工农医类2025版）

本模板是中国海洋大学硕士博士学位论文（理工农医类）LaTeX 模板。
本模板在 [中国海洋大学硕博士学位论文 LaTeX 模板 (2025版)](https://github.com/oucailab/OUC-LaTex-master)
基础上，
根据《中国海洋大学研究生学位论文书写范例（理工农医类）》修改。
